Welcome to the JunkPunk Online Store


The JunkPunk, an online store implemented solely with JavaScript, is a final project for WWW 181 - JavaScript Client Scripting class at Oakton Community College.

The project is organized around a fictional store selling garage sale items called JunkPunk and demonstrates how JavaScript can be used to set up a shopping cart.

The project demonstrates the following application of JavaScript:
JavaScript Enhanced Images
  • Rollovers
  • Banner Ads
  • Rotating Images
Form Validation
  • Registration Form
  • Login Form
  • User Name Form
  • Contact Form
Customized Shopping Cart
  • Objects
  • Functions
  • Cookies
  • Calculations
Other scripts
  • Date Stamp
  • Date Last Modified
  • Pop Up Windows

      A HIDDEN frame is the area of a Web Site that contains JavaScript code which can be accessed from multiple Web pages within a Web site. The hidden frame size is set to zero which makes it invisible for the user. As long as the browser window remains open the data remains intact in the hidden frame as the user navigate a Web Site.

      JavaScript shopping cart code consist of the ShoppingCart object and several functions that allow user to:
      Add item to a shopping cart - "javascript:top.HIDDEN.addtoCart('Yellow Mug', 1.99)"
      View shopping cart - "javascript:top.HIDDEN.viewCart();"
      Delete items from shopping cart - "javascript:top.HIDDEN.deleteItem(0)"
      Do a final checkout - "javascript:top.HIDDEN.checkOut();"
      Checks if the user has logged in by checking cookies
      Offers to login or register
      Display a final checkout page listing user details read in a cookie file and list of items added to a shopping cart with calculated total, tax and grand total.

      A MAIN frame contains html pages with Add to shopping cart links for each of the items in the page. This links execute the addtoCart(); function when the user clicks one of them. The addtoCart(); finction accepts parameters for the item desctiption and the item price.
